Everything You Need

Powerful features to help you manage your IDE workspaces

📁

Multi-IDE Support

Works with both Cursor IDE and Trae IDE. Manage workspaces for your preferred AI-powered IDE.

📂

Workspace Management

Create new workspaces, import existing ones, and manage all your IDE workspaces in one beautiful interface.

🔍

Venomfiles Dependency Checker

Check dependencies for project components with Venomfiles folders. Ensure your projects are properly configured.

Quick Actions

Double-click to open workspaces, right-click for context menu actions, jump to repositories, and edit source files.

Repository Jump

Click to open dependency repositories directly in your browser. Quick access to view source code and documentation.

✏️

Edit Source Files

Edit .rb files and other dependency sources directly in your system default editor.

Refresh Status

Refresh Venomfiles status for all components with one click. Always see the latest dependency state.

Privacy First
